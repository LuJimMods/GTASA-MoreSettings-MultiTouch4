# GTASA MoreSettings MultiTouch4

Plugin AML para GTA San Andreas Android que adiciona suporte aprimorado a multitouch, permitindo o uso de até 4 dedos simultaneamente.

## Recursos

- Suporte para até 4 toques simultâneos.
- Compatível com GTA SA Android v2.00.
- Compatível com AndroidModLoader (AML).
- Integração ao menu MoreSettings.
- Não requer modificação do APK.

## Instalação

1. Instale o AndroidModLoader.
2. Copie o arquivo `.so` para:

```
Android/data/com.rockstargames.gtasa/files/AML/
```

3. Inicie o jogo.

## Como usar

1. Abra o menu AML.
2. Entre em MoreSettings.
3. Ative a opção MultiTouch4.
4. Volte ao jogo.

## Requisitos

- GTA San Andreas Android v2.00
